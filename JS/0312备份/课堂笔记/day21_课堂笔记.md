#### 昨日回顾

```
let和const
	相同点
		都有块级作用域
		没有变量提升(声明之前不能使用)
		在同一个域中不能重复声明
		有暂时性死区
	不同点
		let声明的变量可以修改
		const声明的变量一般是常量
	使用
		顺序:const > let > var
		const一般用于对象类型,let一般用于基本数据类型
		
类和继承
	定义类
		class 类名{
			constructor(参数列表){
				this.属性名 = 参数名
			}
			方法名(方法的参数列表){
				方法体
			}
		}
	继承
		class A extends B
	关键字
		super代表父类,可以直接调用父类的方法(普通方法和构造方法)
	静态
		使用static修饰的,直接使用类名调用

解构赋值
	本质是赋值运算符的扩展
	const [元素列表] = 数组名
	const [属性名列表] = 对象
	
箭头函数
	基本格式 参数=>函数体
	完整格式 (参数)=>{函数体}

遍历总结
	数组遍历的4种方式
		1,普通for循环:速度最快,可读性差
		2,forEach:速度较快,并且可以获取元素和索引,最常用的
		3,for...of:只能获取到元素,获取不到索引,但是可以通过元素获取索引
		4,for...in:只能获取到索引,获取不到元素,但是可以通过索引获取元素
	遍历对象常用的方法
		for...of优势:可以在遍历的同时使用解构的方式快速获取部分数据
		for...in优势:可以很方便的获取对象的所有属性名(后期可以根据属性名获取属性值)
	使用建议
		优先使用forEach,能看懂for...of和for...in
		
三大系列
	offset偏移值
		作用
			获取元素的位置及大小(包含边框和内边距,不包含外边距)
		属性
			offsetTop 上外边距
			offsetLeft 左外边距
			offsetWidth	宽度(包含边框和内边距,不包含外边距)
			offsetHeight 高度(包含边框和内边距,不包含外边距)
	
	client可视区
		作用
			可以动态获取元素边框的大小,宽高
		属性
			clientTop	上边框
			clientLeft	左边框
			clientWidth 内容可视高度(含内边距,不包含边框,外边框和滚动条)
			clientHeight 内容可视宽度(含内边距,不包含边框,外边框和滚动条)
			
	scroll滚动
		作用
			可以动态获取元素滚动的距离,元素大小
		属性
			scrollTop  垂直滚动的距离
			scrollLeft 水平滚动的距离
			scrollWidth	元素的宽度(包括带滚动条的隐藏的地方)
			scrollHeight 元素的高度(包括带滚动条的隐藏的地方)
			
	总结
		offset主要用于获取偏移值,来决定元素的位置
		client主要用于获取元素的可视区域宽高
		scroll主要用于获取元素滚动的距离来做一些判断
```

#### 三大系列注意点

```
1,offset和style区别
	offset
		可以得到任意引入方式中的样式值
		得到的数值是没有单位
		获取的宽高包含边框和内边距
		它是一个只读属性,只能获取不能设置
	style
		只能得到行内样式中设置的属性
		得到的数值带单位
		获取的宽高包不含边框和内边距
		它是一个可读写属性,能获取也能设置
	总结
		想要获取元素的位置和大小,使用offset更方便(offset主要用于获取)
		想要修改元素的属性,使用style更方便(style主要用于设置)
		
2,监听整个页面的滚动距离(监听window的scroll事件)
	pageXOffset 和 pageYOffset 属性
	scrollX 和 scrollY 属性
```

#### 异常

```
概念
	程序出现了不正常的情况(语法错误等),JavaScript 引擎通常会停止，并生成一个错误消息
抛出异常
	系统自动抛出异常
		一般是语法错误,拼写错误,服务器异常,系统会自动停止程序,并且后续代码无法执行
	手动抛出异常
		程序没有问题,程序员根据编程的需要,人为的抛出异常,达到某些目的,后续代码无法执行
处理异常
	目的
		确保程序即使有异常的出现,也能继续执行下去,增强程序的健壮性
	语法
		try {
            ...    //一般会把可能出现问题的代码放到这里,也可以做异常的手动抛出
        } catch(e) {
            ...    //一般书写异常的捕获与处理的代码
        } finally {
            ...    //不管有没有异常,都必须要执行的代码,放到这里(记录日志等操作)
        }
```

#### Promise入门

```
概念
	本质就是js的一个内置对象,对象有属性和方法,它有两个属性和一个方法
	一个状态属性,一个结果属性,还有一个方法then方法
创建
	通过new创建,必须要传一个函数,函数一般有两个形参,resolve和reject
	
属性
	状态属性promiseState
		默认是pending状态(准备状态)
		调用resolve()方法可以将准备状态变成完成状态
		调用reject()方法可以将准备状态变成拒绝状态
	结果属性promiseResult
		默认是undefined(里面没有数据)
		调用resolve和reject方法的时候,传递一个参数,就可以将参数保存到结果属性中

then方法
	接收两个函数类型的参数,如果p的状态是成功的,就会调用第一个函数,如果是失败的,调用第二个函数
	then方法必须在p的状态发生改变的时候才会调用
	给then方法里面的回调函数添加一个参数,就可以拿到结果属性中的值
	
then方法返回值
	返回的是一个promise对象,可以实现链式编程
```

#### async和await

```
概念
	async:异步的意思,可以将同步函数变成异步函数
	await:等待的意思,等待它后面的promise执行完成之后,拿到结果属性中的值
作用
	可以简化调用者获取promise结果的过程
```

##### 页面元素的动画效果

```
实现步骤
	1,在html页面找到要添加动画的元素,加上data-reveal属性,值为bottom,left,right
	2,在css页面编写页面元素的初始样式和复位样式
	3,在js中获取所有要添加动画的元素
	4,监听页面的加载事件,在页面加载完成之后,遍历所有的动画元素,添加复位样式
	5,监听页面的滚动事件,在事件处理函数中,添加判断,当元素距离可视区域小于一屏的时候才复位
```
